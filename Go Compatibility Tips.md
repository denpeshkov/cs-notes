# Overview

Use an unexported, [zero-width](Go%20Zero-Sized%20Values.md), non-comparable field to prevent clients from comparing a `struct`:

```go
type S struct {
	_ [0]func()
	...
}
```

Another way is to define a `nocmp` type and embed it into structs:

```go
type nocmp [0]func()

type S struct {
	nocmp
	...
}
```
